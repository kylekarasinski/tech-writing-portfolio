Java Text Adventure Game



Developed as a semester-long project for my Java course, this text-based adventure game introduced me to game development and is my largest project to date. Throughout the process, I gained valuable experience in encapsulation, debugging, and handling many pieces of code efficiently. Additionally, I focused on enhancing the user experience by optimizing the interface and ensuring that vital information, such as the player's current location and available commands, was clear and easy to access.
